# Company-Employees-Management-System

一个基于C++和Qt（图形化提供）的练习类的封装小型项目

本软件有以下功能：
1. 添加员工信息<br>
2. 查询员工信息和工资展示<br>
3. 编辑员工信息<br>
4. 删除员工<br>
5. 将员工信息以employees.txt文件存储<br>
6. 从employees.txt文件读取记录<br>

使用方法：
CMD命令行版本（version v1.0）
通过输入数字进入对应功能<br>
保存到的文件将会保存在代码文件目录的employees.txt中<br>

QT图形化版本（version v2.0）
通过点击上方的功能选择添加或编辑功能<br>
在下一行输入员工的类型、编号、姓名、等级、工作天数后点击右侧添加/编辑即可<br>
查询员工先选择按编号还是按姓名，输入相关信息即可在下方信息栏显示相关的成员信息<br>
点击exit退出程序

工资计算（纯虚构）：
经理-每日100<br>
技术人员-每日90<br>
销售人员-每日80<br>
销售经理-每日70<br>
等级A-工资x1.5<br>
等级B-工资x1.2<br>
等级C-工资x1<br>

设计非常的简陋和普通，皆在练习C++中类的封装和QT图形化，可用于开源学习<br>

English version<br>
A small encapsulated project of exercise class based on c++and QT (provided graphically)
This software has the following functions:
1. add employee information<br>
2. query employee information and salary display<br>
3. edit employee information<br>
4. delete employee<br>
5. store employee information as employees.txt file<br>
6. read records from the employees.txt file<br>

Usage:
CMD command line version (version v1.0)
Enter the corresponding function by entering a number<br>
The saved file will be saved in employees.txt in the code file directory<br>

QT graphical Version (version V2.0)
Click the function above to select Add or edit functions<br>
Enter the type, number, name, grade and working days of the employee in the next line and click Add/edit on the right<br>
To query employees, first select by number or by name, and then enter relevant information to display relevant member information in the information column below<br>
Click exit to exit the program

Salary calculation (pure fiction):
Manager - daily 100<br>
Technicians -90 per day<br>
Salesperson - daily 80<br>
Sales Manager - daily 70<br>
Grade a-salary x1.5<br>
Grade b-salary x1.2<br>
Grade c-salary x1<br>
The design is very simple and common. They are practicing class encapsulation and QT graphing in c++and can be used for open source learning
